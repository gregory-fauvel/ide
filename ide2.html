Rj Text ed

RJ TextEd est un éditeur de code source et de texte Unicode gratuit pour Windows, qui peut également être utilisé comme simple outil de développement Web. L'éditeur utilise diverses techniques pour la coloration syntaxique dans le source. Il peut utiliser l'auto-complétion et des astuces pour aider à l'édition de code source. Les aperçus de code HTML / ASP / PHP sont pris en charge. Un éditeur de fichier de syntaxe est inclus. [1] L'interface est basée sur le MDI avec des onglets pour l'édition de plusieurs fichiers et la manipulation de documents ouverte. TextEd comprend un navigateur Web , un gestionnaire de fichiers et un éditeur CSS , ainsi que divers outils pour les développeurs Web Outils Ajout d'un nouveau type pour envoyer des demandes POST. Il est utilisé pour publier le document actuel ou une sélection sur une adresse Web et ouvrir le texte renvoyé dans un nouvel onglet de document. Mettre en valeur les expressions Les expressions ajoutées sont maintenant stockées entre les sessions. Une case à cocher permet d'activer une expression à mettre en évidence pour le fichier en cours. Les expressions en surbrillance actives sont désormais stockées entre les sessions pour chaque fichier. Vous pouvez activer / désactiver cette option dans les options, avec d'autres options de fichier telles que l'état de retour à la ligne, le cryptage, l'état de repliement du code, etc.
