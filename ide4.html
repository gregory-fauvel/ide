Light Table


Montres
Next-gen println pour garder une trace des valeurs importantes de votre code. Ajoutez simplement une montre à une expression et la valeur sera renvoyée à LT en temps réel.
ÉVALUATION EN LIGNE
Plus besoin d'imprimer sur la console pour voir vos résultats. Évaluez simplement votre code et les résultats seront affichés en ligne.
OPEN SOURCE
Les outils de développement doivent être open source. Chaque partie du code de Light Table est disponible pour la communauté car aucun d'entre nous n'est aussi intelligent que nous tous
MALLÉABLE
Les cartes clés, les comportements et le moteur Behavior-Object-Tag de Light Table vous permettent de façonner facilement votre IDE en fonction de vos besoins.
GESTIONNAIRE DE PLUGINS
Le gestionnaire de plug-ins se connecte à la liste centrale des plug-ins pour que vous n'ayez pas à chercher partout sur Internet pour trouver ceux que vous voulez
ÉDITION PUISSANTE
Light Table est une interface légère, élégante et élégante, dotée de toute la puissance et de toutes les fonctionnalités dont vous avez besoin et que vous attendez de votre éditeur
